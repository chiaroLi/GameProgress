# GameProgress
A progress for game. To find out which charactor should appear on the stage.

最近玩了个游戏，阴阳师。对战模式是标准的回合对战制。对里面角色的行动顺序很感兴趣。

里面每一个角色都有一个速度值，根据速度值来计算出手顺序。

基本上每个角色的速度都是140左右，行动条总值据说是200。

因此，使用一些基本的数据，想写了一个算法，计算出手顺序。
